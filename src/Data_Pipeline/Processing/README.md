# Data Processing

This directory contains all data processing components.

## Components

- **Parsing/**: Document parsing (optional)
- **Chunking/**: Document chunking into smaller pieces
- **Metadata_Extraction/**: Metadata extraction (optional)
- **Embedding/**: Vector embedding generation

## Processing Order

1. Parsing (if needed)
2. Chunking
3. Metadata Extraction (if needed)
4. Embedding
