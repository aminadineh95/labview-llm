# Destination Connector

Components for storing processed data to various destinations.

## Supported Destinations

- **Vector Stores**: Pinecone, Weaviate, Chroma, etc.
- **Databases**: PostgreSQL, MongoDB, etc.
- **File Systems**: Local file storage (JSON, CSV, etc.)

## Components

- Vector Store Connector VI
- Database Connector VI
- File System Connector VI

## Usage

Configure the appropriate connector based on your storage requirements.
